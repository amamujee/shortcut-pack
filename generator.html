Import on Mac

  1. Open System Settings, then go to Keyboard.
  2. Open Text Replacements.
  3. Drag in Text Substitutions.plist.

Why this works

This uses the import format macOS already supports directly, so it is simpler and more durable than writing into hidden system stores.

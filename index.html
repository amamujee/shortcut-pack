FAQ

Questions, answered.

Is it free?

Yes. The builder and starter pack are free to use.

Do you save my data?

No. Your personal text is not collected or stored by us.

Can I customize it?

Yes. The starter pack is only the starting point.

Can I use it offline?

Yes. Download the builder file and use the same flow locally in one HTML file.

Can I set this up from iPhone or iPad?

No. The plist import step works on Mac only.

Import on Mac

  1. Open System Settings.
  2. Go to Keyboard, then open Text Replacements.
  3. Drag the downloaded Text Substitutions.plist into the list.

Why this route

This generator creates the same plist format that macOS supports importing directly, which is safer than writing to hidden internal stores that can change between macOS versions.

Apple Sync

Set it up once on your Mac.

Shortcut Pack uses Apple’s own text replacements, so the same shortcuts can work on your Mac, iPhone, and iPad.

1

Build it in the browser.

Add the details you reuse. Pick which shortcuts to keep. Rename anything.

>nmFull name
>emEmail
>homeHome address
>introIntro reply
2

Import once on your Mac.

Find Text Replacements in System Settings, then drag Text Substitutions.plist onto it.

3

Use it everywhere.

Use the same shortcuts on Mac, iPhone, and iPad with iCloud sync.

FAQ

Questions, answered.

Less typing · Less remembering · More speed
Is it free?

Yes. The builder and starter pack are free to use.

Do you save my data?

No. Your personal text is not collected or stored by us.

Can I customize it?

Yes. The starter pack is only the starting point.

Can I use it offline?

Yes. Download the builder file and use the same flow locally in one HTML file.

Can I set this up from iPhone or iPad?

No. The plist import step works on Mac only, then Apple can sync the replacements to your other devices.
